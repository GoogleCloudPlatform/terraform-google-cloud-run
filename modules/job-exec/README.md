# Cloud Run Job

## Description

### tagline

Deploy a Cloud Run Job and execute it.

### detailed

This module was deploys a Cloud Run Job run and executes it.

## Usage

Basic usage of this module is as follows:

```hcl
module "job-exec" {
  source = "GoogleCloudPlatform/cloud-run/google//modules/job-exec"
  # Locked to 0.20, allows minor updates â€“ check for latest version
  version = "~> 0.23"

  project_id = var.project_id
  name       = "simple-job"
  location   = "us-central1"
  image      = "us-docker.pkg.dev/cloudrun/container/job"
  exec       = true
}
```

Functional examples are included in the
[examples](./examples/) directory.
